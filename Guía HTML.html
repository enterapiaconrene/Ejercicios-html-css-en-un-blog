Listas

La etiqueta < ul > se usa para listas no ordenadas.

La etiqueta < ol > se usa para listas ordenadas.

La etiqueta < li > se usa para cada elemento de una lista ordenada o no ordenada.

Como usar las tablas

La etiqueta de < tr > o table row, se usa para especificar las filas de nuestra tabla.
La etiqueta de < th > o table head, se usa para especificar la cabecera.
La etiqueta de < td > o table data, define elementos estandares de nuestra tabla.
La etiqueta < thead > define el espacio de cabecera.
La etiqueta < tbody > define el cuerpo de la tabla.
La etiqueta < tfoot > define el pie de la tabla
